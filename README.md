## Task 2: Student Analysis
**Files:** `Student.java`, `StudentMain.java`, `studentsEN.txt`

A system that reads data from a text file and performs analysis:
1. **Model:** `Student` class with fields (name, surname, score), constructor, getters/setters, and `toString`.
2. **Main:** `StudentMain` class reads `studentsEN.txt` into an `ArrayList` and runs streams:
   - **Filter:** Print students with score 5.
   - **Average:** Calculate average score.
   - **Count:** Count students with score 5.
